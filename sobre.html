Sobre a Fake Store API

FakeStoreAPI é uma REST API gratuita que fornece dados fictícios para projetos de e-commerce ou testes de aplicações, sem necessidade de backend. Ideal para ensino, exemplos de código, prototipagem e testes.

Exemplo de código

fetch('https://fakestoreapi.com/products/1')
  .then(res => res.json())
  .then(json => console.log(json));

Recursos disponíveis

  • Products: 20 produtos
  • Cart: 20 itens
  • Users: 10 usuários
  • Login: JWT Token

Rotas disponíveis

  • GET /products
  • GET /products/1
  • GET /products/categories
  • GET /products/category/jewelery
  • GET /products?limit=5
  • GET /carts?userId=1
  • POST /products
  • PUT /products/1
  • PATCH /products/1
  • DELETE /products/1
